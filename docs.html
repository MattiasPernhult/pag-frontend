PAG Mashup Documentation

General

This documentation describes how to use the PAG API which is a mashup project. The mashup combine data from different RESTful API sources and these are The Movie Database, Snooth, Brewery DB and Snacks (our own RESTful API which is not available for public use).

Response format

All the response are formated in JSON

/movie

Example

api/tips/movie

/snack

Required parameters

movie_genre = A string containing one or more genres seperated by a “+” character. Notice that the genre must start with a capital letter.

Optional Parameters

kind = An integer value representing which snack to generate.

Example

api/tips/snack?movie_genre=Adventure
api/tips/snack?movie_genre=Animation+Thriller
